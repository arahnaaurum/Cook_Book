Cook Book (книга рецептов)
Краткое описание:
Фуллстек-приложение для просмотра рецептов блюд, использующее:
    1) для бэкенда - Django Rest Framework, автодокументацию OpenAPI+Swagger
    2) для фронтенда - React (без подключения Redux)

Полное описание:
1) Бэкенд (/cookbook) - сервер на Django, запускается на localhost:8000
    БД - по умолчанию (SQLite)
    URL для просмотра документациии OpenAPI - http://localhost:8000/swagger-ui/

2) Фронтенд (/cookbook_app) - React-приложение, запускается на localhost:3000
    (!) Поскольку приложение небольшое, redux в данном случае не подключен, а элементы при необходимости обращаются к серверу через хук useState.

3) Пользовательский интерфейс:
    Рецепты распределены по категориям. У каждого блюда и каждой категории есть своя страница.
    С главной страницы можно перейти на любую из категорий, а из категории — на любой рецепт этой категории.
